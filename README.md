### 🚀 What I'm building

**[Reposcope](#)** — An AI-powered codebase onboarding agent that helps developers understand unfamiliar repos in minutes. Generates dependency graphs, a Code Health Score, tech-stack detection, and Q&A over the codebase using a LangChain supervisor pattern with Groq + Chroma, served over FastAPI/SSE with a React frontend.

**[MockMate AI](#)** — A LangChain-powered interview prep bot that simulates real interview flows with contextual follow-ups.

**[HireLens](#)** — A resume screener that returns structured candidate evaluations using Pydantic-enforced LLM outputs.

**[ResearchAgent](#)** — A two-agent research pipeline (Tavily + BeautifulSoup) that streams findings live over SSE.

**[QueryVault](#)** — A RAG pipeline built from scratch with metadata filtering and HyDE for sharper retrieval.
